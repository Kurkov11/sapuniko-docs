The Challenge

The client requested a modern, minimalist eCommerce website for an electronics business. It should also support blogging.

We will approach this website with utility-first approach, avoiding unnecessary clutter. We should direct user‚Äôs attention to the offered products and seamlessly incorporate the blog. A toned down color palette will be good to maintain the minimalist approach. We will also add a highlight color that will be associated with electronics.

Solution

The design featured a clean grid layout with ample white space, using light gray (#F5F5F5) and white (#FFFFFF) with dark gray (#333333) text for readability. A subtle highlight color, either blue (#00AEEF) reinforced the tech theme.

Products were displayed in a clean, card-based layout, with blog articles seamlessly integrated into the homepage.
